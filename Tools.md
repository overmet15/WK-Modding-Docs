# Tools
List of tools used for mod development.

### [BepInEx 5](https://github.com/bepinex/bepinex/releases/latest)
BepInEx 5 is LTS/Stable version of BepInEx, there is BepInEx 6 but its still in development.  
With it, you are able to run the mods, atleast the ones that were made for BepInEx.

### [Unity Explorer](https://github.com/sinai-dev/UnityExplorer)
Very usefull tool for checking and playing with objects and values in the game.  
It was used to gain info about the UI and other objects.
